# ai-news-system

AI 新闻聚合站（静态可部署版本），接入 8 类来源并生成自动总结：

1. OpenAI Blog
2. Anthropic News
3. GitHub Trending AI
4. Product Hunt AI
5. AI Startup Funding
6. YouTube AI 频道
7. X/Twitter AI KOL

自动总结包含：
- 发生了什么
- 为什么重要
- 普通人机会在哪

## 本地预览

```bash
python3 -m http.server 8000
```

打开 `http://localhost:8000`。
